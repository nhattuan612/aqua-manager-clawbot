# 11. Page Patterns

Tài liệu này mô tả các pattern trang chuẩn cho hệ UI AQUA.

## 11.1 Overview Page

### Dùng cho
- màn tổng quan điều hành
- landing page sau login

### Cấu trúc
1. title + subtitle
2. dense-stat summary
3. group chips
4. alert cards
5. log row / activity row

### Mục tiêu
- nhìn 5 giây phải biết toàn hệ đang thế nào

## 11.2 Inventory Page

### Dùng cho
- nhiệm vụ
- service
- timer
- package
- skill

### Cấu trúc
1. summary
2. filter/sort row
3. group sections
4. card grid hoặc table

### Quy tắc
- object nhiều ngữ cảnh -> card
- object thiên so sánh -> table

## 11.3 Knowledge Page

### Dùng cho
- soul
- memory
- learning
- user profile
- docs nội bộ

### Cấu trúc
1. summary
2. group strip
3. grouped cards
4. modal đọc full content

## 11.4 Logs Page

### Dùng cho
- activity stream
- log per process
- audit action

### Cấu trúc
1. summary count
2. source filter
3. expandable log rows

### Quy tắc
- summary đọc trước
- raw log mở sau

## 11.5 Settings Page

### Dùng cho
- config
- auth
- policy
- toggle hệ thống

### Cấu trúc
1. title + giải thích ngắn
2. grouped sections
3. field labels rõ
4. primary action cuối section

## 11.6 Helper / Access Page

### Dùng cho
- tunnel
- command
- tokenized access
- route phụ

### Cấu trúc
1. quick notes
2. command boxes
3. route items

### Quy tắc
- ít prose
- nhiều action rõ ràng

## 11.7 Danger / Backup Page

### Dùng cho
- backup
- restore
- delete package

### Cấu trúc
1. summary
2. grouped cards
3. action buttons rõ mức độ

### Quy tắc
- destructive action phải tách biệt
- backup/restore phải nhìn ra ngay
