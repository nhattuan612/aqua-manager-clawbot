# 13. Iconography

Tài liệu này chuẩn hoá icon và emoji trong hệ UI AQUA.

## 13.1 Vai trò icon
Icon trong AQUA dùng để:
- định hướng nhanh
- phân loại vùng nội dung
- làm mềm giao diện vận hành

Icon không dùng để:
- thay thế text
- trang trí ngẫu nhiên

## 13.2 Icon cho tab
- `🏠` Tổng Quan
- `🚀` Nhiệm Vụ
- `🧩` Skill Forge
- `⏰` Nhắc Việc
- `🧠` Tàng Thư Các
- `📦` Gói Github
- `🛡️` Sao Lưu
- `🔑` Khoá/API
- `💬` Quyền Telegram
- `🖥️` Cổng OpenClaw

## 13.3 Icon cho group
- `🖧` system service
- `⏲️` timer
- `🌐` gateway
- `🤖` agents
- `📘` learning
- `🫀` soul
- `👤` user
- `💓` heartbeat

## 13.4 Icon cho card type
- `🔑` token
- `🧪` API chưa dùng
- `📄` code/file
- `🗂️` snapshot
- `💬` telegram group
- `👤` actor/admin/user
- `🛡️` backup target

## 13.5 Icon cho trạng thái
- `★` đang theo dõi
- `☆` chưa theo dõi
- `●` trạng thái chấm màu

## 13.6 Quy tắc dùng
- một vùng chính chỉ cần 1 icon chủ đạo
- không trộn 2-3 icon cạnh nhau nếu không cần
- icon phải nhất quán giữa các màn

## 13.7 Khi nào không dùng icon
- bảng dày dữ liệu kỹ thuật
- badge đã đủ nghĩa
- nơi icon làm rối hơn là giúp

## 13.8 Nếu cần icon mới
Icon mới phải:
- đơn nghĩa
- không xung đột icon cũ
- phù hợp ngữ cảnh vận hành
